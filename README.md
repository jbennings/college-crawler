# College Crawler

A Python console application that reads a spreadsheet of community college districts and colleges, crawls official websites, extracts key business development data points, and exports an enriched Excel database.

## Project Goals

- Read college and district data from Excel
- Find official websites
- Identify leadership, board, directory, contact, and strategic plan pages
- Extract useful data points
- Assign confidence scores
- Export results for human review
